<h1 align="center">Árvore de Links</h1>

### 🔖 Sobre

Esse site tem como objetivo reunir os principais links necessários para realizar contato comigo, foi desenvolvido em ReactJs como forma de estudo.

---

### 🏁 Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com) e o [Node](https://nodejs.org/en/).

### 🎲 Rodando
- Clone este repositório <br>
```git clone https://github.com/Casmei/link-tree.git```

- Acesse a pasta do projeto no terminal/cmd <br>
```cd link-tree```
- Instale as dependências do projeto <br>
```npm install```
- Inicialize inicialize o projeto com o comando <br>```npm run dev```
